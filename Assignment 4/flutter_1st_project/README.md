Flutter UI Clone - Static Screens
================================

This is a ready-made Flutter project (static UI) that recreates the 3-screen dark marble interface.

How to run:
1. Unzip the project.
2. Open the folder in VS Code or Android Studio.
3. Run `flutter pub get`.
4. Run on your device/emulator: `flutter run`.

Assets included:
- assets/wallpaper.png  (dark marble wallpaper)
- assets/icons/*        (simple placeholder icons)

Note: This is a static UI mockup. No interactive functions are used.
